# ヘッダーをコンパクト化＆少し濃いめの青に戻す

## 変更点

**色の調整**
- ヘッダーとメッセージ入力エリアの背景を黄色から元の青系に戻します
- 以前の青よりほんの少しだけ濃いめのトーンに調整して、より引き締まった印象に

**ヘッダーのレイアウト修正**
- グループ名（「サボり防止部」）とメンバー数の行を上に詰めて、アイコン一覧と被らないようにします
- 上部のセーフエリア（時計・電池表示の下）の余白を削り、グループ名がもっと上に来るように
- グループ名行とアイコン一覧行の間隔も最小限にして、ヘッダー全体をさらに薄く
- 結果としてタイムライン領域がさらに広くなります

ステータスバー（時計・電波・電池）の背景も引き続きヘッダーの青に揃えて、自然に繋がるようにします。
