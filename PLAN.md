# 勉強開始ボタンをタイムライン領域の右上に移動・背景色調整

**変更内容**

- ナビゲーションバー右上にある勉強開始ボタン（再生ボタン）を、タイムライン領域の右上端（メンバーアイコン一覧の右側に重ねる形）に移動します。
- ボタンが上に消えた分、メンバーアイコン一覧をグループ名のすぐ下、ぎりぎりまで上に詰めます。
- メンバーアイコンのサイズをほんの少しだけ小さくします（40 → 36）。
- タイムラインタブの「メンバーアイコン一覧エリア」の背景は、他のタブと同じ通常の背景色（systemGroupedBackground）に統一します。
- 投稿が並ぶタイムライン領域の背景は、よりはっきり区別できるよう一段暗いグレーに変更します。
