# チャットの100文字制限と画面タップでキーボードを閉じる


## 変更内容

- **コメント（チャット）の文字数制限を100文字に**
  - タイムライン画面のチャット入力と、グループチャット画面のチャット入力の両方で、100文字を超える入力をブロック
  - 入力欄の右下に「文字数 / 100」のカウンターを表示し、上限近くで色が変わる
  - 100文字を超えた状態では送信ボタンを無効化

- **画面のどこかをタップでキーボードを閉じる**
  - タイムライン画面とグループチャット画面で、入力欄以外の任意の場所をタップするとキーボードが閉じる
  - メッセージ一覧をスクロールしたときも、これまで通りスワイプでキーボードを閉じられる挙動は維持
