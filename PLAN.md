# 撮影間隔を重み1:2:2:1の4分割分布に変更

## 変更内容

撮影間隔のランダム抽選を、現在の一様分布から、中央が出やすい分布に変更します。

- **範囲**: 1秒 〜 最大値（平均×2−1秒）※今までと同じ
- **4分割**: 範囲を4等分
- **重み**: 最初の区間 = 1、2番目 = 2、3番目 = 2、最後の区間 = 1
- **挙動**: 各区間が選ばれる確率が 1:2:2:1 になり、選ばれた区間内では一様にランダム
- **平均**: 重みが左右対称なので、平均時間は今までと変わりません

これにより、極端に短い・長い間隔は出にくくなり、平均付近の間隔が出やすくなります。3つのモード（スキマ時間／通常／長時間）すべてに適用されます。
