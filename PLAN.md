# 投稿カードにモード（スキマ時間／通常／長時間）を表示

## やること

タイムラインの投稿カードに、その勉強セッションがどのモード（スキマ時間／通常／長時間）で行われたかを表示します。

## 表示内容

- 投稿カードの時間・経過時間の隣に、モードのアイコンと名称をチップで表示します
  - スキマ時間: 時計アイコン + 「スキマ時間」
  - 通常: 本アイコン + 「通常」
  - 長時間: 炎アイコン + 「長時間」
- チップは小さめで、淡い背景色のカプセル型。モードごとに色を変えて視覚的に区別します（スキマ時間=ティール、通常=ブルー、長時間=オレンジ）

## 対応範囲

- 新規投稿時にモード情報も保存されるようにします
- 既存の（モード情報がない）投稿はチップを非表示にして、これまで通り表示されます
- 下書きから投稿した場合も、下書きに保存されたモードがそのまま反映されます

## 影響しないもの

- ユーザー個人の勉強時間・セッションデータ
- チャット、グループ、承認機能
