# 撮影モードに「開発モード」追加と外出時モードの展開UI化

## 変更内容

### 撮影モードの展開UI

- 撮影モードのカードをタップすると、そのカードのすぐ下に「外出時モード」トグルがアニメーションで現れる
- 外出時モードのカードは撮影モードのカードより一回り小さめのデザイン（コンパクトなアイコン・余白）
- 説明文を「カフェや図書館など、周囲に配慮したい場所でおすすめ」に変更
- 既定はオフのまま。一度タップで開き、もう一度同じカードをタップで閉じる
- 画面上部の独立した「外出時モード」セクションは削除（カードの下に統合）

### 開発モード（新規追加）

- 撮影モード一覧に「開発モード」を追加
- サブタイトル：「平均10秒」
- アイコン：`hammer.fill`（開発っぽさを表現）
- カラー：紫系
- 撮影間隔：おおよそ平均10秒になるよう短く設定（テスト・動作確認用）

## 動作イメージ

1. ユーザーがモード設定画面を開く
2. スキマ時間 / 通常 / 長時間 / 開発モード の4枚のカードが並ぶ
3. 一枚タップすると選択され、そのカード直下に小さな「外出時モード」トグルがにゅっと出現
4. 別カードをタップすると、外出時モードトグルがそちらへ移動
