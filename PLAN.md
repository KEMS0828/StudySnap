# タイムラインのデザイン刷新＋キーボード＋グループ参加時のチャット表示

## 変更内容

### 1. 「勉強を始める」ボタンの刷新

- 現在の細長いリスト行風デザインから、**存在感のあるヒーローボタン**に変更
- 青→シアンのグラデーション背景、柔らかい影、再生アイコン入りの大きな円形バッジ
- 「勉強を始める」を太く大きく、その下に「カメラで記録して仲間に承認してもらおう」のサブコピー
- タップ時に軽く縮むアニメーションとハプティクス
- 制限中のときはロック＋アップグレード訴求のカードに切り替え（今より見やすく）

### 2. 投稿カードのデザイン刷新

- 角丸を大きくして柔らかい影を追加し、カードに立体感を出す
- ヘッダー：アバター周りに薄いリング、名前を太字、時間・勉強時間を小さなピル型タグに
- 承認ステータスを右上の**小さなバッジ**に整理（全承認＝緑のチェックシール、部分承認＝青のピル、未承認＝オレンジのピル）
- 勉強内容（科目）を**強調されたタグ風ラベル**（薄い青背景＋本アイコン）
- 振り返りテキストに余白とラインハイトを追加して読みやすく
- 写真の下に薄い区切り線を入れて情報ブロックを整理
- 全体的に余白・文字サイズを調整してApple純正アプリ風の洗練された見た目に

### 3. 投稿の編集画面：キーボード外タップで閉じる

- テキストフィールド以外をタップするとキーボードが閉じる機能を追加
- 既存の「完了」ボタンとスクロールで閉じる挙動はそのまま維持

### 4. グループ参加時にチャット（コメント）も表示

- グループに参加した直後、投稿は読み込まれるが**チャットメッセージが読み込まれていない**不具合を修正
- 参加処理の中でチャット履歴も一緒に取得するようにする
- これで参加前の投稿とコメントの両方がタイムラインに時系列で並ぶ

## 影響する画面

- タイムライン画面（勉強を始めるカード、投稿カード）
- 投稿編集画面（キーボード挙動）
- グループ参加時の裏側処理（チャット読み込み追加）
