# モザイクの強度をストロークごとに調整できるようにする

## 変更内容

写真編集画面のモザイクツールに「強度」スライダーを追加し、描く前にモザイクの粗さを調整できるようにします。

### 機能

- モザイクツール選択時、ブラシサイズスライダーの下に「強度」スライダーが表示される
- 強度を低くすると細かいモザイク、高くすると粗い（はっきり隠す）モザイクになる
- スライダーで設定した強度は、その後に描くストロークに適用される
- すでに描いたモザイクはそれぞれ描いた時の強度で保持されるので、1枚の写真の中で部分ごとに強度を変えられる
- 下書き保存しても強度の情報は保持される

### 画面・操作

- 写真編集画面のモザイクツールツールバー：
  - 1段目：ブラシサイズスライダー（既存のまま）
  - 2段目：モザイク強度スライダー（新規追加、左に小さい四角アイコン・右に大きい四角アイコン）
