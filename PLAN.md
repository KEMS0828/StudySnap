# タイムライン画面のヘッダーと入力欄を整える

## 改善内容

### ヘッダーをコンパクトに

- グループ名アイコンの上下余白を詰めて、ヘッダー全体の高さを縮小
- メンバーアバター列のサイズと余白を少し小さくして、タイムライン表示領域を広げる
- 「メンバー○人」「○人が勉強中」などの行間も詰めて密度を上げる

### ステータスバー（時計・電池）の背景を統一

- ヘッダーの淡い水色グラデーション背景を、画面上端のステータスバー部分まで延長
- 時計・アンテナ・電池表示の背景がヘッダーと同じ色になり、段差や違和感がなくなる

### メッセージ入力欄をヘッダーと統一感のあるデザインに

- 入力バーの背景を、ヘッダーと同じ淡いブルー系のトーンに変更
- 送信ボタンはヘッダーの再生ボタンと揃ったブルー〜シアンのグラデーションに
- 入力欄上部に細い区切り線を追加して、タイムラインとの境界を明確化

### ヘッダーとタイムラインの境目をはっきり

- タイムライン側の背景を少しだけ濃いめのグレーに調整
- ヘッダー下端の区切り線をもう少し濃く・はっきり見えるように
- 「ヘッダー（淡いブルー） → 区切り線 → タイムライン（グレー）」の段差で境界がしっかり認識できる

### 全体の見た目

- 画面上部から下部まで、ブルー基調で統一感のある一枚絵に
- ヘッダー・本文・入力欄の3エリアが、それぞれ境界線でくっきり分かれる
- 投稿カードを表示できる領域が今より広くなる
