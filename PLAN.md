# タイムラインのヘッダーを大人っぽくリフレッシュ

他タブとの統一感を保ちつつ、ヘッダー周りに上質さと情報の見やすさを足します。配色は今のブルー×シアンを継承します。

## 改良ポイント

### ヘッダー全体
- 上部に薄いブルー→シアンの**サブトルなグラデーション帯**を敷き、その上に半透明の素材（ぼかしガラス調）を重ねて奥行きを出す
- 下端に細いディバイダーを追加して投稿エリアと自然に区切る
- 全体の高さを少しだけ広げ、要素間に余白を確保して窮屈さを解消

### グループ名（タイトル行）
- グループ名の左に**小さな円形アイコン**（青×シアンのグラデ円に「人が3人」のシンボル）を添え、識別感を強化
- グループ名の下に「メンバー○人 ・ 今○人が勉強中」の**サブテキスト**を小さく追加（赤いドット付き）
- タップ可能であることが分かるよう、シェブロンを少し控えめな配置に調整

### メンバー一覧
- アバターを少し大きく（34pt → 38pt）し、勉強中の赤いリングをよりはっきり、外側にうっすら**赤いグロー（光彩）**を追加して視認性アップ
- 勉強中のメンバーは下に小さな「●勉強中」ラベルか、アバター右下に脈打つドット
- メンバー名（名前のイニシャルや短縮）をアバター下に小さく表示するオプション
- 勉強中のメンバーが**先頭に並び替え**されて目立つように

### スタートボタン
- 今の青×シアンの円形ボタンは維持しつつ、サイズを少し大きく（52pt → 56pt）
- 周囲に**やわらかい呼吸する光（パルスアニメーション）**を追加して「押せる感」を強調
- 押すと軽く跳ねるスプリングアニメーションとハプティクス
- ロック状態（無料枠終了時）はゴールド寄りのグラデーションに変えて「Pro案内」感を出す

### 細かい仕上げ
- ヘッダー登場時にメンバーアバターが**順番にふわっと現れる**ステージドアニメーション
- スクロールでヘッダーが少しコンパクトに縮む（タイトルが小さくなる）演出も検討
- ダークモードでも違和感が出ないようグラデの透明度を調整

他タブ（ホーム、プロフィール等）との統一感を壊さないよう、背景色とカード形状の基調はそのまま残し、ヘッダー部分だけを上質にアップグレードする方針です。
