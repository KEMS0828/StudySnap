# モザイクのスライダー範囲調整＆開発用10秒モード追加

**モザイクのデフォルトを中央に**

- モザイクの強度スライダーの最大値を下げて、現在のデフォルトがちょうど真ん中に来るように調整（5〜15、デフォルト10）
- モザイクのブラシサイズの最大値も下げて、現在のデフォルトが真ん中に来るように調整（20〜40、デフォルト30）
- 投稿編集画面と写真編集画面の両方に同じ調整を適用

**開発用モードの追加**

- モード選択に「開発用」モード（平均撮影間隔10秒）を追加
- 既存の3モード（スキマ時間 / 通常 / 長時間）の並びの最後に表示
- 撮影間隔の確率分布（4分割で真ん中2つが2倍）はそのまま適用
