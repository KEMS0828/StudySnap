# タイムラインで自由にチャットを送れるようにする（NGワード対応）

## 機能

- タイムライン下部の固定メッセージ（クイックメッセージ）バーを、自由に文章を入力できるテキスト入力欄に置き換えます
- 送信したメッセージは今までと同様にタイムラインに混ざって時系列で表示されます（チャット吹き出し形式）
- 同じグループのメンバー間でのみ表示されます（既存のチャット機能と同じ範囲）
- 送信ボタンは入力中のときだけ青く有効化され、空文字では送信できません
- 1回の送信は最大200文字までに制限します
- 送信中はキーボード上に入力欄が留まり、送信後に自動でクリアされます

## NGワードフィルター

- 不適切な日本語ワード（暴言・差別語・性的表現・誹謗中傷など）を内蔵リストとして持ちます
- メッセージ本文にNGワードが含まれている場合、送信をブロックして警告アラートを表示します（「不適切な表現が含まれているため送信できません」）
- 表記ゆれ（ひらがな・カタカナ・全角半角・記号挟み）にもある程度対応するため、判定前にテキストを正規化します
- クイックメッセージ送信機能は廃止します（自由入力で代替できるため）

## デザイン

- タイムライン下部に、丸みのある入力欄＋右側に紙飛行機アイコンの送信ボタンという iMessage / LINE 風の入力バー
- 入力欄は半透明素材（.bar）の上に配置し、薄いグレーの角丸テキストフィールド
- 送信ボタンはアクティブ時に青、未入力時はグレーアウト
- キーボード表示時に入力欄が自然に持ち上がる（ネイティブ挙動）
- NGワード検出時はシンプルな警告アラートを表示
