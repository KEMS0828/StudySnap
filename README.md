# StudySnap

勉強を“証明”するアプリ。

ランダム撮影 × 他人承認で、
サボれない勉強習慣を作る。

-「家で勉強しようとしてもスマホ触っちゃう」

-「気づいたら寝てた」

そんな人のために、
"ちょうどいい周りの目" を作るアプリです。

## 特徴
- ランダム撮影でサボれない
- 写真をグループで承認
- 勉強時間を自動記録
- 努力を可視化

## 経緯
受験期時代、学校・塾では勉強できていたが、家では集中できずに困っていました。

そんな時に、気軽に・適度な「周りの目」を作れないかなと思っていたのが経緯です。
勉強記録アプリは多いが、
「本当にやったか」を証明できるものは少ない。
そこでこのアプリを制作しました。
